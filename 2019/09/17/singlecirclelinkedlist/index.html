Josephu问题
Josephu问题为：设编号为 1，2，… n的 n个人围坐一圈，约定编号为 k（1<=k<=n）的人从 1开始报数，数到m的那个人出列，它的下一位又从1开始报数，数到m的那个人又出列，依次类推，直到所有人出列为止，由此产生一个出队编号的序列。
提示：
用一个不带头结点的循环链表来处理 Josephu问题：先构成一个有 n个结点的单循环链表，然后由 k结点起从 1开始计数，计到 m时，对应结点从链表中删除，然后再从被删除结点的下一个结点又从 1开始计数，直到最后一个结点从链表中删除算法结束。

约瑟夫问题-创建环形链表的思路图解

代码如下：

Boy boyFirst = null;//需要一个头
 public void addBoy(int num) {
        Boy curBoy = null;
        if(num<1){
            System.out.println("小孩数量不正确");
            return;
        }
        for (int i = 1; i <= num; i++) {
            Boy tb=new Boy(i);
            if (i == 1) {
                boyFirst=tb;            //将第一个节点赋给头节点
                boyFirst.next=boyFirst; //构成环形链表
                curBoy=boyFirst;        //curBoy记录当前节点位置
            }else{
                curBoy.next=tb;         //将curBoy指向刚创建好的节点
                tb.next=boyFirst;       //构成环形链表
                curBoy=tb;              //curBoy移动到当前节点
            }

        }
    }

约瑟夫问题-小孩出圈的思路分析图

Boy boyFirst = null;
    /**
     *
     * @param firstNum 从第几个节点开始数
     * @param passNum  数几个节点
     * @param num       一共有几个节点
     */
    public void outCircle(int firstNum,int passNum,int num){
        if(firstNum<1||firstNum>num){
            System.out.println("数的方式不规范");
            return;
        }
        Boy firstBoy = boyFirst;
        Boy helper = null;
        while(true){
            if(firstBoy.next!=boyFirst){
                firstBoy=firstBoy.next;
            }else {
                helper = firstBoy;
                firstBoy = firstBoy.next;
                break;
            }
        }
        for(int i=1;i<firstNum;i++){
            firstBoy=firstBoy.next;
            helper=helper.next;
        }
        while(true) {
            if(firstBoy==helper){
                break;
            }
            for (int i = 1; i < passNum; i++) {
                firstBoy = firstBoy.next;
                helper = helper.next;
            }
            System.out.println("先出来的小孩是"+ firstBoy.getNo()+" 号");
            firstBoy = firstBoy.next;//将与要出圈的节点相关联的节点指向变到相关节点上就是删除该节点了。
            helper.next = firstBoy;
        }
        System.out.println("最后在圈里的小孩是"+ firstBoy.getNo()+" 号");
    }
